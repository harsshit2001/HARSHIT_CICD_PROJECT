## 📌 Project Overview
This project demonstrates a simple **CI/CD pipeline** using GitHub along with a deployed portfolio website.

The website is built using:
- HTML
- CSS
- JavaScript

and is automatically deployed using **GitHub Pages**.

---

## ⚙️ Features
- 🌍 Live deployed website
- 🎨 Responsive design using CSS
- ⚡ Interactive elements using JavaScript
- 🔄 Continuous Deployment via GitHub Pages

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript  
- Git & GitHub  
- GitHub Actions (CI/CD)

---

## 📂 Project Structure

---

## 🚀 How It Works (CI/CD)
1. Code is pushed to GitHub repository  
2. GitHub Actions workflow is triggered  
3. Website is automatically deployed  
4. Live site is updated instantly
