# 🚀 CI/CD Pipeline using GitHub Actions

## 📌 Project Overview
This project demonstrates a simple CI/CD pipeline using GitHub Actions to automatically deploy a static website.

## ⚙️ Technologies Used
- GitHub
- GitHub Actions
- HTML

## 🔄 CI/CD Workflow
1. Code is pushed to the main branch
2. GitHub Actions workflow is triggered
3. The project is built and deployed automatically
4. Website is hosted using GitHub Pages
